# Rainfall OOP Final Project

Java Object-Oriented Programming final project that analyzes rainfall data for 12 months across 3 years using a `Rainfall` class.

## Project Overview

This project was created for CPEN 358 - Object-Oriented Programming. The purpose of the program is to store and analyze rainfall data for three years. Each year contains rainfall values for the 12 months of the year.

The rainfall data is stored in a 2D (two-dimensional) array of `double` values. The program uses OOP (Object-Oriented Programming) principles by creating a `Rainfall` class with attributes, constructors, getters/setters, and methods that perform the required calculations.

## Features

The program can perform the following operations:

- Calculate the total rainfall for the three years
- Calculate the average monthly rainfall for the three years
- Calculate the total rainfall for each year
- Determine the year with the most rain
- Determine the year with the least rain
- Determine the month with the most rain in a specific year selected by the user
- Determine the month with the least rain in a specific year selected by the user
- Demonstrate the functionality of the `Rainfall` class through a complete driver program
